# Project 'Any Cocktail'

This is a simple SPA which main purpose is to search for recipies of cocktails. Deployed on Firebase. [Try it out here](https://any-cocktail.web.app/cocktails?s=berry).

## Technologies and tools used

- React
- React Redux
- React Router

## API

[Cocktails API](https://rapidapi.com/yadahan/api/cocktails1/) found on RapidAPI

## The idea

I came up with this idea while learning React and exploring the free API's in the RapidAPI. This project was the way for me to practice what I've learnt so far and also now I have a great tool so search for new cocktail recipies üçπ
